Stereopix viewer examples - Conversion from Suto's viewer

« Return to index

This example shows how to send a list of images originally created for the "HTML5 viewer" of M.Suto to the viewer of Stereopix.

The conversion is done (and transmitted) by script. It works both with the list transmitted through the pics variable in the old versions and through the img1 variable of most recent versions (if you remove the new ThView part). For the second syntax, this converter only works with the 'p' entries yet.
